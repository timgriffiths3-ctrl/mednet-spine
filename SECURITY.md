# Security Policy

If you discover a security vulnerability, please report it responsibly.

## Reporting

- Preferred: Open a confidential security report to the repository maintainers or contact via email: security@mednet.one (replace with a real address or GitHub security contact).
- If you require encryption for sensitive details, provide a PGP key or use GitHub's private security advisory flow.

## Response process

- We will acknowledge receipt within 72 hours and provide an estimated timeline for a fix.
- We will coordinate disclosure and provide guidance for updating customers if needed.

## Important notes

- Do NOT include any raw biometric data
